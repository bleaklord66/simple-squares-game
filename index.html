Rules

The aim of this game is to get the chest with the minimum number of rolls. First of all you need to roll the dice. Than you can move based on what number you rolled. You cannot go to the squares you already been to. You can get the chest only if you will have zero steps left after you step on it.

Controls

Move with WASD or with arrows. Roll with the button or with R. To restart the game, click to the restart button. To espace this menu, use either the espace key or the X symbol.
